Multi-stage SaaS sales pipeline

1 · Self-serve signupLifecycle stage, source, ICP segment, product intent, trial start, MQL score.
2 · QualifiedRequired: persona, company size, use case, ARR estimate, next step owner.
3 · Demo / discoveryMeeting booked, pain category, decision process, competitor, timeline.
4 · Proposal / enterprisePlan tier, stakeholders, legal/security status, forecast category, close date.
5 · Won / onboardingHand-off checklist, onboarding segment, success owner, activation event.

Webhook/API workflow handling

Error handlingLog failed deliveries, alert owner, create task, re-enroll only after fix.visible

Custom behavioral event setup

Event

● `activation_completed` for in-app action, not a page view/form fill.

Properties

Plan, account ID, role, feature used, timestamp, workspace size.

Use

Segment activated trials, trigger sales task, suppress nurture, route enterprise users.
