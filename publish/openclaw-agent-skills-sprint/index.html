Project-specific OpenClaw preview

Two custom OpenClaw skills: viral scout + client research briefer

A lean sprint plan for setting up an OpenClaw AI agent system with Node.js/Claude API integration, focused on two practical skills: a daily viral content format scout and an on-demand client research briefer.

OpenClawNode.jsClaude APIDaily digestResearch automation

Skill 1: Viral content format scout

1

Monitor selected social/content sources and extract format patterns: hooks, structure, audience, CTA, repeatability.

2

Deduplicate weak trends, score formats by usefulness, and produce a concise morning digest.

3

Output ready-to-test content formats, not generic “trend news”.

Skill 2: Client research briefer

1

Triggered on demand with company/person/domain/context.

2

Collect relevant facts, business model, current positioning, likely pains, and outreach angles.

3

Return a clean brief with source notes and recommended next action.

Implementation plan

Day 1: confirm OpenClaw setup, runtime, skill schema, secrets, and Claude API config.
Day 2: build both skill skeletons, prompts, input/output schemas, and test fixtures.
Day 3: connect source adapters, digest formatting, error handling, and handoff docs.
Day 4: QA with sample runs, edge cases, and final installation notes.

Key focus: keep the agents useful, debuggable, and easy to operate — not over-engineered.
