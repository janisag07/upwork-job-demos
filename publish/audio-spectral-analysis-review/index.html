Reference track intake

Each track gets a structured row instead of vague audio notes.

  • file/source label
  • track purpose: breathwork, entrainment, voice bed, transition
  • first-pass issue tags
  • sample-task ready export

Spectral + dynamics review

Focus on technical differences the client can act on.

  • low/mid/high band balance
  • RMS/LUFS consistency notes
  • harshness/masking hotspots
  • stereo width and movement

Production pipeline recommendations

Turns analysis into a decision document.

  • which sonic traits to copy or avoid
  • safe volume/headroom targets
  • voice/music balance guidance
  • handoff notes for producer

Paid sample task structure

Keeps the first step small and reviewable.

  • analyze 1–2 tracks first
  • deliver annotated report
  • confirm format
  • then complete remaining references
