1Emulator extraction map

Define selectors, scroll logic, retries, screenshots/logs, and safe run limits for the Android/Appium layer.

2Structured listing parser

Normalize title, price, year, mileage, location, seller and source metadata into a predictable JSON schema.

3Dedupe + QA queue

Prevent duplicate vehicles, flag missing fields, detect price anomalies, and keep failed records reviewable.

4Node / Next.js handoff

Expose a clean API payload for scheduled processing, dashboards, alerts, or downstream scoring logic.
