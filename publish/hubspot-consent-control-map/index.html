Consent architecture map

1

Form consent field

Separate explicit marketing opt-in from general contact/request submission.

2

Subscription type

Map checkbox language to HubSpot subscription types, not one global yes/no.

3

Marketing contact status

Only promote contacts when opt-in and lawful basis are present.

4

Workflow guardrails

Prevent automation from subscribing contacts after normal form submissions.

5

Audit trail

Store source form, consent copy, timestamp, and rollback path.

Interactive scenario

IF form.newsletter_opt_in = true\nTHEN set subscription: Marketing Newsletter = Subscribed\nAND set marketing contact = yes\nLOG consent source + text version + timestamp
IF form.newsletter_opt_in is empty\nTHEN create/update contact for request handling\nDO NOT subscribe to marketing\nDO NOT enroll in nurture workflow
IF contact unsubscribes\nTHEN suppress all marketing workflows\nKEEP service/transactional communication separate\nLOG opt-out event
