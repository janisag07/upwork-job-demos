# Demo Plan — Upwork Super 5 — 2026-07-07

## 1. Action-Safe Support Agent Console
- Slug: `publish/action-safe-support-agent-console/index.html`
- Demo concept: Reads a customer request, selects the safe internal tool, calls mocked order/status/ticket APIs, and escalates when data is missing or sensitive.
- First milestone: map request types, internal API schemas, missing-data rules, and human-escalation boundaries before building the first reliable agent loop.

## 2. Real Estate Speed-to-Lead Command Center
- Slug: `publish/real-estate-ghl-speed-to-lead-system/index.html`
- Demo concept: New Zillow/Realtor/Facebook lead arrives → GHL responds within 60 seconds → SMS bot qualifies → calendar booking → missed-call recovery and voice handoff are scoped safely.
- First milestone: map lead sources, fields, calendar rules, SMS templates, and account/phone dependencies, then build the first speed-to-lead workflow with test leads.

## 3. Medical Device Multichannel Outreach Control Map
- Slug: `publish/medical-device-multichannel-outreach-map/index.html`
- Demo concept: Physician list cleanup in Zoho, segmented outreach paths, human sales-agent handoff, and channel-level tracking for a hybrid AI + human medical-device sales system.
- First milestone: run an architecture sprint: define Zoho fields, segmentation, channel ownership, compliance boundaries, and the first measurable campaign before scaling the full 2–3 week build.

## 4. HubSpot CRM Automation Health Map
- Slug: `publish/hubspot-crm-automation-health-map/index.html`
- Demo concept: Pipeline, lifecycle stage, custom property, workflow, data cleanup, integration, and dashboard checks organized into a clear first diagnostic sprint.
- First milestone: start with a compact HubSpot audit: current pipelines, stages, properties, automations, integrations, and the one workflow/dashboard that should be cleaned first.

## 5. Meta Lead Forms → Lovable CRM Router
- Slug: `publish/meta-forms-to-lovable-crm-router/index.html`
- Demo concept: Meta instant-form payload is captured, validated, deduped, mapped into Lovable CRM fields, and verified with a test-lead log before handoff.
- First milestone: confirm Meta form access, Lovable CRM destination fields/API path, then wire one test lead end-to-end and document the mapping.
